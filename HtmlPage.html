רקע ותיאור הפרויקט

מדוע בחרתי בנושא?

בחרתי בנושא שיתוף מזון קהילתי כי הוא משלב שתי בעיות בוערות בישראל של היום: יוקר המחייה ובזבוז המזון העצום. בישראל נזרקות מדי שנה מאות אלפי טונות מזון לפח, בעוד משפחות רבות מתקשות לעמוד בהוצאות האוכל. האתר שלי נותן מענה לשתי הבעיות בבת אחת.

קהל היעד

  • אנשים שיש להם מזון עודף שפג תוקפו יפוג בקרוב — ורוצים לשתף במקום לזרוק.
  • משפחות ויחידים שמחפשים מוצרי מזון בחינם באזורם.
  • ארגוני צדקה וגמ"חים המעוניינים לרכז תרומות מזון.

מטרות האתר

  • הפחתת כמות המזון הנזרק לפח.
  • יצירת קהילה תומכת המשתפת מוצרים ברחבי הארץ.
  • עזרה לאוכלוסיות שמתקשות כלכלית.
  • מתן כלי חיפוש ופילטור נוח לפי עיר, קטגוריה ותאריך תפוגה.

שאילתות SQL מרכזיות

1. הצגת כל המוצרים (JOIN) + חישוב ימים שנותרו:

SELECT FoodItems.ItemID, FoodItems.UserID, FoodItems.ItemName, FoodItems.Category, FoodItems.PickupCity, FoodItems.PickupLocation, FoodItems.ExpiryDate, FoodItems.Quantity, Users.UserName, DateDiff('d', Date(), FoodItems.ExpiryDate) AS DaysLeft FROM FoodItems INNER JOIN Users ON FoodItems.UserID = Users.id WHERE FoodItems.ExpiryDate >= Date() ORDER BY FoodItems.ExpiryDate ASC;

2. חיפוש מסונן לפי עיר, קטגוריה וזמן תפוגה:

SELECT ... FROM FoodItems INNER JOIN Users ON FoodItems.UserID = Users.id WHERE FoodItems.ExpiryDate >= Date() AND FoodItems.PickupCity = 'Tel Aviv' AND FoodItems.Category = 'פחמימות' AND FoodItems.ExpiryDate <= DateAdd('d', 7, Date()) ORDER BY FoodItems.ExpiryDate ASC;

3. הוספת מוצר (INSERT):

INSERT INTO FoodItems (UserID, ItemName, ExpiryDate, Quantity, Category, PickupCity, PickupLocation, PostDate) VALUES (?, ?, ?, ?, ?, ?, ?, ?);

4. עדכון מוצר (UPDATE):

UPDATE FoodItems SET ItemName=?, ExpiryDate=?, Quantity=?, Category=?, PickupCity=?, PickupLocation=? WHERE ItemID=?;

5. מחיקת מוצר (DELETE):

DELETE FROM FoodItems WHERE ItemID = 5;

6. מחיקת משתמש + כל מוצריו:

DELETE FROM FoodItems WHERE UserID = 12; DELETE FROM Users WHERE id = 12;
